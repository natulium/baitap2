WritePad Editor

Write Pad is an editor application that allows you to create Web pages easily and quickly.

Step to open WritePad application:

  1. Click Start
  2. Click Programs
  3. Click Writepad

Advantages of using a WritePad editor:

  • It allow faster creation of Web pages.
  • It allows easier updation of text or images in WritePad editor.
  • It is platform independent.

Disadvantages of using a WritePad editor.

  • It consumes more memory space.
  • It requires to fill-in more detail such as meta date for each page.
